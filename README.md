## 基于vue+Element-ui+PHP 的后台管理系统

- 前端技术栈：vue
- 后台组件库：Element-ui
- http库：axios (类ajax)
- 后端开发语言：PHP
- 后端开发框架：Yii2
- 数据库服务：Mysql

### 部署

1.cd 到该项目文件夹，命令行运行：composer install,如果环境没有安装composer，请先安装composer  
2.本地建立数据库，数据库名称：demo，账号：root，密码：123456。
3.命令行运行 ./yii migrate 进行数据库迁移，也就是给数据库建表
4. 运行项目命令：./yii serve
